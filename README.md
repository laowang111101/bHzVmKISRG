# 基于SpringBoot+Vue的共享汽车管理系统 java1085

## 项目概述
本项目是一款基于SpringBoot和Vue的共享汽车管理系统，采用前后端分离的设计模式，实现了管理员和用户两种角色的功能需求。

## 技术栈
### 系统架构
- B/S架构
- MVC模式

### 系统环境
- 兼容Windows和macOS操作系统

### 推荐运行环境
- JDK 1.8
- Maven
- MySQL
- Node.js

### 服务端技术
- Spring Boot
- Mybatis-Plus

### 前端技术
- HTML
- CSS
- Layui
- jQuery
- Bootstrap
- Vue
- Axios
- Element-UI

## 功能模块
### 管理员角色功能模块
- 个人中心
  - 查看和修改个人信息
- 用户管理
  - 查看用户信息
  - 添加、修改、删除用户信息
- 投放地区管理
  - 添加、修改、删除投放地区信息
- 汽车信息管理
  - 添加、修改、删除汽车信息
- 汽车投放管理
  - 管理汽车的投放信息，包括投放地区、投放地点、投放时间等
- 汽车入库管理
  - 管理汽车的入库信息，包括回收地点、回收时间等
- 使用订单管理
  - 查看和管理用户的使用订单，包括订单状态、支付情况等
- 汽车归还管理
  - 查看和管理用户的汽车归还信息，包括归还时间、归还内容等

### 用户角色功能模块
- 个人中心
  - 查看和修改个人信息
- 汽车投放
  - 查看可用的汽车投放信息，选择并使用汽车
- 使用订单管理
  - 查看自己的使用订单，进行支付操作
- 汽车归还
  - 查看自己使用的汽车归还信息，完成归还操作

## 核心亮点
- 前后端分离的设计，易于维护和扩展
- 一站式的服务，涵盖源码、数据库文件和设计报告
- 系统功能全面，满足管理员和用户的基本需求
